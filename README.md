## Reactを用いたSOLID原則に基づいたTODOリストの例

関連記事：[React(TypeScript)でSOLIDの原則を心で理解する](https://zenn.dev/andmorefine/articles/d9ca963da8e59a)

## インストール

**Yarn**

Yarnのインストール

```
npm install -g yarn
```

依存関係のインストール

```
yarn install
```

## プロジェクトの起動

```
yarn dev
```
